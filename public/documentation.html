Function Request Method URL Endpoint
Return a list of ALL movies GET /movies
Return data about a single movie by title GET /movies/:name
Return data about a genre by title GET /movies/:name/genre
Return data about a director by name GET /directors/:name
Allow new users to register POST /users
Allow users to update their username PUT /users/:username
Allow users to add a movie to their favorites list POST /users/:username/favorites
Allow users to remove a movie from their favorites list DELETE /users/:username/favorites
Allow current users to deregister DELETE /users/:username
